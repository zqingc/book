小程序引入 Vant Weapp 



1、微信开发者工具创建小程序模板

2、cd 到小程序根目录执行

```kotlin
第一步：npm init

第二步：npm install --production

第三步：npm i vant-weapp -S --production
```

3、在微信开发工具执行npm构建，构建完成后即可引入组件
